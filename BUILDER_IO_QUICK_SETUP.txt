=== НАСТРОЙКИ ДЛЯ BUILDER.IO ===

Скопируйте и вставьте эти настройки в Builder.io:

---

Environment Variables:
VITE_PORT=3006
VITE_VERSION=2.5.8
VITE_BASE_URL=/
VITE_API_URL=/api/mock
VITE_BUILDER_API_KEY=your-builder-api-key-here
VITE_BUILDER_PREVIEW_URL=http://localhost:3006
NODE_ENV=development

---

Setup Command:
pnpm install

---

Dev Command (ВАЖНО!):
pnpm dev --host 0.0.0.0 --port 3006

---

Package Manager:
pnpm

---

Preview URL Template:
http://localhost:3006$PATHNAME

---

Design Mode Selector:
#app

---

⚠️ ВАЖНО: 
- Dev Command ОБЯЗАТЕЛЬНО должна быть: pnpm dev --host 0.0.0.0 --port 3006
- БЕЗ флага --host 0.0.0.0 сайт НЕ откроется в Builder.io!
- Замените your-builder-api-key-here на ваш реальный API ключ
